# Getting Started

Install Vue with Vite

npm create vite@latest my-vue-app --template vue
cd my-vue-app
npm i
1
2
3

Install d3.js

npm install d3
1

Import everything from d3 as d3 in your Component. e.g. App.vue

<script setup>
import * as d3 from 'd3';
d3.scaleLinear(...)
</script>
1
2
3
4

... or load just load what you need (recommended). For example scaleLinear Import the

<script setup>
import { scaleLinear } from 'd3';
scaleLinear(...)
</script>
